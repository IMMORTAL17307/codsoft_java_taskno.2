# Student Grade Calculator 📝

### 👨‍💻 Developer Details
* **Name:** Ishant Khare
* **Company:** CodSoft

### 🎯 Objective
To develop a Java application that accepts student marks across multiple subjects, calculates the total and average percentage, and assigns a final letter grade.

### 🛠️ Tools Used
* Java
* Arrays and Control Structures

### 📝 Steps Performed
1. Utilized Java Scanner to dynamically collect marks for various subjects.
2. Applied mathematical logic to sum the marks and compute the exact percentage.
3. Used `if/else` control structures to map the percentage to a standard letter grade.

### 💡 Outcome
An accurate, automated grading tool that processes multiple data points to output a clear performance summary.
